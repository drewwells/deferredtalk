Why are we using RequireJS?

Why are we using RequireJS?

Why not labjs, YepNope.js, ControlJS, Load.js, etc?

  • labjs: Blocking
  • YepNope: Conditional
  • RequireJS: module loader

Why are we using RequireJS?

  • CommonJS and AMD support
  • Optimized script loading
  • Modularized code
  • Avoid one JS file with 8,000 LOC
  • Optimized built file

Where are we going?

Where are we going?

  • One giant, unreadable JavaScript file
  • Abstracted, reusable and maintainable code
  • JavaScript apps
  • Segmentation of features/functionality
  • Code ownership
  • Less is more, faster site performance

Caveats

  • Avoid comments before the define definition.
  • Cross dependencies a->b, b->a
  • One module per file
  • Plugins: text.js, order.js, r.js
  • Always specify dependencies
